### DEfination
	Mo rong port bang chot
		Khi chot = 1: co the dua du lieu ra output
		Khi chot = 0: du input thay doi thi out push cung khong thay doi

### Pins
	D0-D7	input data pins
	Q0-Q7	output data pins
	OE		output enable (active LOW)
	LE		latch enable
